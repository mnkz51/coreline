# Core-Line Project Specification

## 1. 概要
- 正式名称: Core-Line (コアライン)
- ジャンル: 戦略・自動化・ファンタジー
- コンセプト: ヘクスグリッド上にマナの導線を築き、領地を最適化する

## 2. 技術的制約
- 描画: PixiJS (CDN経由で読み込み)
- グリッド: Red Blob Games式の「Cube Coordinates」を自作実装（外部ライブラリ禁止）
- 実行環境: ブラウザ直接実行（JavaScript ES Modules）
- レスポンシブ: スマホの全画面表示に対応させる

## 3. 現在の目標 (Step 1: グリッドの展開)
- [ ] 画面いっぱいにヘクスグリッド（六角形）を表示する。
- [ ] 各ヘクスの中央に座標 (q, r, s) をデバッグ表示する。
- [ ] ヘクスをタップ/クリックすると、そのヘクスが強調表示（ハイライト）される。
- [ ] 画面リサイズ時に描画領域が追従するようにする。
